# Modal

To import the `Modal` component simply include the following at the top of your file:

```
import { Modal } from '@dicoauto/toolbox';
```

## With actions

<Canvas>
  <Story name="With actions">
    {() => {
      const [opened, set] = useState(true);
      return (
        <div>
          {!opened && (
            <button
              className="text-toolbox-primary underline font-semibold leading-5 text-sm"
              type="button"
              onClick={() => set(true)}
            >
              Reopen the modal here
            </button>
          )}
          {opened && (
            <Modal
              title="Welcome"
              acknowledgement="Yes"
              dismissal="Skip for now"
              onAcknowledge={() => set(false)}
              onDismiss={() => set(false)}
            >
              See all the new features we have added so far.
            </Modal>
          )}
        </div>
      );
    }}
  </Story>
</Canvas>
