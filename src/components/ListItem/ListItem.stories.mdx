# ListItem

To import the `ListItem` component simply include the following at the top of your file:

```
import { ListItem } from '@dicoauto/toolbox';
```

## Centered Text

In special cases you want to show important information by its own, you can center the title by
using `textAlignment` and setting it to `center`. Doing this will also disable the subtitle if set.

<Canvas>
  <Story name="Centered text">
    <ListItem
      textAlignment="center"
      title="Text label"
      subtitle="Text label"
    />
  </Story>
</Canvas>
