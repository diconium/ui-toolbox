# Toast

To import the `Toast` component simply include the following at the top of your file:

```
import { Toast } from '@dicoauto/toolbox';
```

## With action

<Canvas>
  <Story name="With action">
    {() => {
      const [counter, set] = useState(0);
      return (
        <Toast
          icon
          label="With action"
          subtext={!counter ? 'Press the X' : `onClose called: ${counter}`}
          onClose={() => set(counter + 1)}
        />
      );
    }}
  </Story>
</Canvas>
