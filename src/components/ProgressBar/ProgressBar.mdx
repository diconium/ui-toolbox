# ProgressBar

To import the `ProgressBar` component simply include the following at the top of your file:

```
import { ProgressBar } from '@diconium/ui-toolbox';
```

## Dynamic Min and Max

You can set `min` and `max` options if you want to set the minimum progress
that should be shown. In that case the progress will not drop below the `min` value and not go over the `max` value.
