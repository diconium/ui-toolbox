# Drawer

To import the `Drawer` component simply include the following at the top of your file:

```
import { Drawer } from '@dicoauto/toolbox';
```

## Default

<Canvas>
  <Story name="Default">
    <Drawer />
  </Story>
</Canvas>

## Open and Close

Clicking on the handle on the top will open and close the drawer content.
