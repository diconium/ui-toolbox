# Drawer

To import the `Drawer` component simply include the following at the top of your file:

```
import { Drawer } from '@dicoauto/toolbox';
```

## Default

<Canvas>
  <Story name="Default">
    <Drawer />
  </Story>
</Canvas>

## Open and Close

Clicking on the handle on the top will open and close the drawer content.

<Canvas>
  <Story name="Opened">
    <Drawer opened>
      <span className="text-center py-4 font-thin text-base text-toolbox-neutral">
        Are you really sure?
      </span>
      <PrimaryButton label="Action" />
    </Drawer>
  </Story>
</Canvas>

## Different layouts for the content area

The Drawer components supports two different layouts: `centered`(default) and `full-width` as shown below:
