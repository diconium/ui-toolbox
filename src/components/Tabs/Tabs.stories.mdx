# Tabs

To import the `Tabs` component simply include the following at the top of your file:

```
import { Tabs, Tab } from '@dicoauto/toolbox';
```

## Default

<Canvas>
  <Story name="Default">
    {() => {
      const [selected, select] = useState(0);
      return (
        <Tabs
          selected={selected}
          onSelect={(index) => select(index)}
          tabs={[
            { label: 'All', icon: 'menu' },
            { label: 'New', icon: 'engine-warning' },
            { label: 'Read', icon: 'check' },
            { label: 'Loader', icon: 'battery-bolt' },
          ]}
        />
      );
    }}
  </Story>
</Canvas>

  <Story name="As block component">
