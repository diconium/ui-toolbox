# Select

To import the `Select` component simply include the following at the top of your file:

```
import { Select } from '@dicoauto/toolbox';
```

## Default

<Canvas>
  <Story name="Default">
    {() => {
      const [selected, select] = useState(null);
      return (
        <Select
          selected={selected}
          placeholder="Select one option please"
          options={[
            { id: 1, label: 'Text label 1' },
            { id: 2, label: 'Text label 2' },
            { id: 3, label: 'Text label 3' },
            { id: 4, label: 'Text label 4' },
            { id: 5, label: 'Text label 5' },
          ]}
          onSelect={(selected) => select(selected)}
        />
      );
    }}
  </Story>
</Canvas>
