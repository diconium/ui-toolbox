## Colors

The library comes with a set of colors as shown below. They are available in the
standard tailwindcss nomenclature e.g.

```
[modifier]-toolbox-[primary|secondary|alert]-[name of color]
```

### Examples:

```
bg-toolbox-primary-orange

bg-toolbox-secondary-purple

text-toolbox-alert-green
```
